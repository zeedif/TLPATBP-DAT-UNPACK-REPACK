# THE LIAR PRINCESS AND THE BLIND PRINCE DAT UNPACK-REPACK

Programa y entorno de trabajo para cifrar y/o descifrar el archivo DAT del videojuego *The Liar Princess and the Blind Prince* (PC/Steam).

Solo basta con pegar el contenido de este repositorio en la raíz del juego, abrir el `DAT UNPACK-REPACK.bat` en Windows y seleccionar la opción deseada.
